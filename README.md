#  QuizPulse AI – Smart Quiz & Student Engagement Platform

A full-stack AI-powered quiz platform that generates dynamic quizzes in real-time using Wikipedia + AI, tracks student progress, and provides analytics for both students and teachers.

##  Features

### Student Features
-  Secure Login & Signup with Email OTP verification
-  AI-generated quizzes based on topic input
-  Adaptive difficulty (Easy → Medium → Hard)
-  Timer + Auto Submit
-  Tab switching detection (anti-cheat)
-  Dashboard with:
  - Total attempts
  - Average score
  - Topic
